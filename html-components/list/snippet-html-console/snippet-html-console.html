snippet-html-console

10 Dec 2025

Description

the snippet-html-console, allows running a html snippet with a console output

Include

initmod

ext = params.ext; $ = params.$; menu = params.menu; ace = params.ace; menumod = params.menumod; log = params.log;
name description
ext the default loader helper
$ the dom helper module
menu | menumod either the main menu or the menumod module
ace a global reference to ace editor
log a reference to the global log-mod

API

[attribute]

src
url
the url of the text to load in the editor
also supports the attributes for web-editor

[module]

name
run()
parameters
none
no return value
run the html text
