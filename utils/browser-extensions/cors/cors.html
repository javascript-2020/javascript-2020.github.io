CORS Extension

07 Jan 2026

very simple extension to add cors headers to all requests from all urls, meant for dev purposes, sometimes simple is best.

Please Note : some sites can break when an extension injects cors headers, if a website begins acting strange remember to disable this cors extension. A ui could and may be developed to allow inclusion and exclusion domains, but for now its just meant be simple and easy.
