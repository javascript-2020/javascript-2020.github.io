# life

loosesly based on leina lifa and an extension of conways life, this code snippet demonstrates what can be achieved
by apply simple rules over a complex system.  The rules themselves define things like how the atoms interact
with atoms of their own colour, forinstance yellows slowly tend to move away from each other, greens and reds
tend to be atttracted to their atoms of their own colour.  The rules also define how atoms interact with atoms
of different colours, for instance red be attracted to green, yellow be attracted to green, red and yellow
gently repel.  These 'forces' are summed for each atom and used to calculate a velocity for each atom.
